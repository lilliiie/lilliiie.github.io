Pacman sur amstrad cpc

Pour comprendre le développement d'un jeu video.

Juste pour le plaisir d'apprendre de nouvelles choses. Pour le développement j'utilise Arnold Emulator, qui me permet de simuler l'amstrad cpc pour lire mes fichiers.
Sdcc ( Small Device C Compiler ) pour compiler mon code "C" grâce au fichier .bat qui contient le code nécessaire à la compilation des librairies, des sprites( images de pacman et des fantômes ) et mes differentes sources ( fichier.c ).
Enfin pour la création de la carte j'utilise Tiled téléchargé sur mapeditor. A ce stade les fantômes ne restent pas dans le labyrinthe et pacman ce déplace grâce aux touches A,Q,P,O. Projet a finaliser.
